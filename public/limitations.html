Lex Fanbot Limitations

The fanbot has access to all transcripts available of the podcast. However there are some episodes for which the list of transcripts are not available. This means that the Fanbot (at the moment) is not capable of searching and creating a response for questions regarding these episodes.

No Memory

Due to the high volume of requests this tool receives and to avoid crashes the system does not keeps track of the full conversation history. In technical terms this means that every question you ask will be treated as the first one without including previous requests in the context. I hope to implement chat context in the near future.

Results
